# ➕ Arithmetic Operators in Java

## 🚀 Introduction
किसी भी रियल वर्ल्ड एप्लीकेशन (जैसे- सैलरी सिस्टम, मार्क्स कैलकुलेटर) में मैथमेटिकल कैलकुलेशन करने के लिए **Arithmetic Operators (अरिथमैटिक ऑपरेटर्स)** का इस्तेमाल किया जाता है।
जावा में कुल **5 प्रकार** के अरिथमैटिक ऑपरेटर्स होते हैं:

| सिंबल (Symbol) | नाम (Name) | काम क्या है? (Usage) | उदाहरण (Example) `a=10, b=3` |
|:---:|---|---|---|
| **`+`** | Addition (जोड़) | दो वैल्यूज़ को जोड़ना | `10 + 3 = 13` |
| **`-`** | Subtraction (घटाव) | एक वैल्यू से दूसरी को घटाना | `10 - 3 = 7` |
| **`*`** | Multiplication (गुणा) | दो वैल्यूज़ को गुणा करना | `10 * 3 = 30` |
| **`/`** | Division (भाग) | क्वोशन्ट (Quotient) पाना | `10 / 3 = 3` (चूँकि दोनों `int` हैं) |
| **`%`** | Modulo (रिमाइंडर) | शेषफल (Remainder) पाना | `10 % 3 = 1` |

---

## 🎯 The Most Important Operator: Modulo (`%`)
इंटरव्यू में सबसे ज्यादा पूछा जाने वाला ऑपरेटर **Modulo (%)** है। 
- **काम:** यह भाग (Division) करने के बाद जो **शेषफल (Remainder)** बचता है, वो रिटर्न करता है।
- **रियल-वर्ल्ड यूज़:** किसी नंबर का **Even या Odd** चेक करने के लिए इसका सबसे ज्यादा यूज़ होता है (उदा: `num % 2 == 0` मतलब इवन नंबर)।
- *Example:* अगर आप 12 को 5 से डिवाइड करेंगे, तो रिमाइंडर (शेषफल) 2 बचेगा। तो `12 % 5` का आउटपुट `2` होगा।

---

## 🚧 Division Trap (Integer vs Double)
जावा में डिवीज़न करते हुए बिगिनर्स अक्सर एक ग़लती करते हैं:
1. `int / int = int`: अगर आप `7 / 2` करेंगे, तो आउटपुट **`3`** आएगा (न कि `3.5`), क्योंकि पॉइंट वैल्यू हटा (Truncate) दी जाती है।
2. `double / int = double`: अगर आपको एक्यूरेट पॉइंट वैल्यू चाहिए, तो कम से कम एक नंबर को `double` या `float` बनाना पड़ेगा। जैसे `7.0 / 2` करेंगे तो आउटपुट **`3.5`** आएगा।

## ⚖️ Operator Precedence (BODMAS Rule in Java)
अगर एक ही लाइन में कई सारे ऑपरेटर्स हों (जैसे `10 + 5 * 2`), तो जावा पहले किसे सॉल्व करेगा? इसे तय करने के लिए **Operator Precedence (प्रायोरिटी)** का रूल लागू होता है।

1. **High Priority (हाई प्रायोरिटी):** `*` (गुणा), `/` (भाग), और `%` (मॉड्यूलो)
2. **Low Priority (लो प्रायोरिटी):** `+` (जोड़) और `-` (घटाव)

### 📌 Example 1: Without Brackets
```java
int result = 10 + 5 * 2;
```
- यहाँ `*` की प्रायोरिटी हाई है, तो पहले `5 * 2 = 10` सॉल्व होगा।
- फिर उसमें 10 ऐड होगा: `10 + 10 = 20`।
- **आउटपुट:** `20`

### 📌 Example 2: With Brackets
अगर हम चाहते हैं कि प्लस (`+`) पहले हो, तो हम ब्रैकेट्स `()` का इस्तेमाल कर सकते हैं। 
```java
int result = (10 + 5) * 2;
```
- ब्रैकेट सबसे हाई प्रायोरिटी पर होता है! पहले `(10 + 5)` सॉल्व होकर `15` बनेगा।
- फिर `15 * 2` होगा।
- **आउटपुट:** `30`
