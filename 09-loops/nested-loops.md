# 🪆 Nested `for` Loop: Loop Inside a Loop

## 🤔 What is a Nested Loop?
जब भी हमें "काम के अंदर काम" (Task inside a Task) करना हो, तो हम नेस्टेड लूप्स का इस्तेमाल करते हैं। 
साधारण शब्दों में: **जब एक लूप के बॉडी (block) के अंदर एक और लूप लिखा जाता है, उसे Nested Loop कहते हैं।**

अगर आप सिर्फ एक लूप का इस्तेमाल कर रहे हैं, तो आप प्रोग्रामिंग का सिर्फ आधा (Half) पावर यूज कर रहे हैं! Pattern printing, 2D Arrays (Matrix), और Game logic—बिना नेस्टेड लूप्स के ये पॉसिबल ही नहीं हैं।

---

## 🗓️ The "Days and Tasks" Analogy
इसको एक रियल-टाइम एग्जांपल से समझते हैं:
मान लीजिये आपके पास टोटल **5 दिन (Days)** हैं, और आपको हर दिन **3 काम (Tasks)** करने हैं।

अगर आप सिर्फ एक नॉर्मल फॉर लूप लगाएंगे, तो आप या तो सिर्फ "दिन" गिन पाएंगे, या सिर्फ "टास्क"। 
लेकिन अगर आप दिन-वाइज टास्क गिनना चाहते हैं, तो:
- **Outer Loop:** यह आपके "Days" (1 से 5) को हैंडल करेगा।
- **Inner Loop:** यह आपके "Tasks" (1 से 3) को हैंडल करेगा।

---

## 💻 Syntax & Internal Working

### Syntax
```java
for (int i = 1; i <= 5; i++) {       // Outer Loop (Days)
    for (int j = 1; j <= 3; j++) {   // Inner Loop (Tasks)
        System.out.println("Day: " + i + "  Task: " + j);
    }
}
```

### 🔄 Internal Execution Flow (काम कैसे करता है?)
1. **Outer Initialization:** बाहर वाला लूप स्टार्ट होता है (`i = 1`).
2. **Outer Condition:** चेक होता है कि क्या `i <= 5` है? (हाँ).
3. **Inner Loop Starts:** अब प्रोग्राम अंदर आ जाता है, और इनर लूप स्टार्ट होता है (`j = 1`).
4. **Inner Loop Cycle:** इनर लूप अपनी *पूरी* साइकिल खत्म करेगा (`j = 1`, फिर `j = 2`, फिर `j = 3`).
5. **Inner Loop Ends:** जैसे ही `j` की वैल्यू 4 होगी, इनर लूप की कंडीशन फाल्स हो जाएगी और वो लूप खत्म हो जायेगा।
6. **Outer Updation:** इनर लूप खत्म होने के बाद, कंट्रोल वापस आउटर लूप के पास जाएगा और `i` को इंक्रीमेंट करेगा (`i++` यानी `i = 2`).
7. **Repeat:** यह पूरी प्रोसेस तब तक चलेगी जब तक आउटर लूप पूरी तरह से खत्म (False) नहीं हो जाता।

> **Golden Rule:** आउटर लूप के एक (1) इटरेशन (Iteration) के लिए, इनर लूप अपनी पूरी (Full) साइकिल कम्पलीट करता है।

## 🎯 Real Use Cases (हम इसका यूज़ कहाँ करते हैं?)
1. **Pattern Printing:** जैसे Stars (*) से Pyramid या Square बनाना।
2. **Matrix (2D Arrays):** रो (Row) और कॉलम (Column) वाले डेटा स्ट्रक्चर्स (टेबल्स) के लिए।
3. **Game Development:** 2D बोर्ड गेम्स या ग्रिड-बेस्ड लॉजिक लिखने के लिए।
