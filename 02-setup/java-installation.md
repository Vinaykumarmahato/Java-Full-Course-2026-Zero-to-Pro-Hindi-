# 🧠 Topic: Java Install & Setup 2026 (JDK + VS Code)

## 🚀 Introduction
- **आज का टारगेट:** इस चैप्टर में हम सीखेंगे कि अपने लैपटॉप या पीसी (PC) में **JDK कैसे इनस्टॉल करें**, **VS Code कैसे सेटअप करें** और अपना **पहला Java प्रोग्राम** (Hello World) सक्सेसफुली कैसे रन करें। 
- **Verified Sources Only:** Java को पहले 'Sun Microsystems' ने बनाया था, लेकिन बाद में इसे **Oracle** ने एक्वायर (Acquire) कर लिया। इसलिए हम JDK को बिलकुल वेरीफाइड और ऑफिसियल वेबसाइट (Oracle) से ही डाउनलोड करेंगे।

## 🛠️ Step 1: Downloading & Installing JDK
1. ब्राउज़र ओपन करें और **Oracle Java** सर्च करें।
2. ऑफिशियल वेबसाइट (`oracle.com/java`) पर जाएं और **Download Java** पर क्लिक करें।
3. **कौनसा वर्ज़न (Version) चुनें?** अभी लेटेस्ट वर्ज़न JDK 25 है, लेकिन एक बिगिनर (Beginner) के तौर पर हम **JDK 21** रिकमेन्ड करते हैं। लेटेस्ट वर्ज़न में बार-बार नए अपडेट्स आते हैं जो कन्फ्यूज़ कर सकते हैं, इसलिए स्टेबल वर्ज़न (21) से शुरुआत करना बेस्ट है।
4. जहाँ आपका OS (Windows/Mac/Linux) लिखा है उसपर क्लिक करके `.exe` फाइल डाउनलोड करें (याद रखें, हर OS के लिए JDK/JVM अलग-अलग होता है)।
5. डाउनलोड की गई `.exe` फाइल पर डबल क्लिक करें और 'Next ➡️ Next' करके C Drive में इनस्टॉल कर लें।

## ⚙️ Step 2: Setting up Environment Variables (Path)
हमारे कंप्यूटर (सिस्टम) को यह बताना बहुत ज़रूरी है कि हमने JDK इनस्टॉल कर लिया है। इसके लिए हमें 'Path' सेट करना होता है:
1. अपने फाइल मैनेजर में जाएं: `C Drive ➡️ Program Files ➡️ Java ➡️ jdk-21 ➡️ bin`
2. सबसे ऊपर एड्रेस बार (Address Bar) पर क्लिक करें और उस पाथ (Path) को **Copy** कर लें।
3. Windows स्टार्ट बटन पर क्लिक करें और **Environment Variables** सर्च करके उसे ओपन करें।
4. **Environment Variables** वाले बटन पर क्लिक करें ➡️ लिस्ट में से **Path** पर डबल-क्लिक करें ➡️ **New** पर क्लिक करें।
5. जो पाथ कॉपी किया था उसे यहाँ **Paste** कर दें और 3 बार **OK** पर क्लिक करके बाहर आ जाएं।
6. ✅ *Verify कैसे करें?* CMD (Command Prompt) ओपन करें और `java -version` टाइप करके Enter दबाएं। अगर आपके सामने Java का वर्ज़न आ गया, तो इसका मतलब JDK परफ़ेक्टली इंस्टॉल हो गया है!

## 💻 Step 3: Installing VS Code (IDE) & Extensions
1. ब्राउज़र में **VS Code download** सर्च करें और `code.visualstudio.com` से Windows के लिए `.exe` फाइल डाउनलोड करें।
2. फाइल ओपन करें और नॉर्मल सॉफ्टवेयर की तरह (Next ➡️ Next ➡️ Finish) इनस्टॉल कर लें।
3. VS Code को ओपन करें।
4. 🔌 **Important Extensions:** लेफ्ट साइड मेनू में 'Extensions' वाले आइकॉन पर क्लिक करें और सर्च करें:
   - **Extension Pack for Java** (Microsoft की तरफ से) - इसे इंस्टॉल करें (यह Java कोड को रन और सपोर्ट करने के लिए सबसे ज़रूरी है)।
   - **Code Runner** - एक क्लिक में कोड रन करने के लिए।

## 🏃‍♂️ Step 4: Running Your First Java Program
1. डेस्कटॉप पर `Java Series` नाम का एक नया फोल्डर बनाएं।
2. VS Code में जाकर, `File ➡️ Open Folder` पर क्लिक करें और इस 'Java Series' फोल्डर को ओपन कर लें।
3. फोल्डर के अंदर नए फाइल वाले आइकॉन पर क्लिक करें और नाम टईप करें: `demo.java` (Java फाइल का एक्सटेंशन हमेशा `.java` होता है)।
4. अपना पहला कोड (Hello World) टाइप करें।
5. **Run कैसे करें?** कोड वाली स्क्रीन पर कहीं भी రైट-क्लिक (Right Click) करें और **Run Code** पर क्लिक करें (या ऊपर दिए गए 'Play/Run' बटन का उपयोग करें)। 
6. टर्मिनल (Terminal) में आपको अपना पहला आउटपुट `Hello World` दिख जाएगा! 🎉 
   *(ध्यान दें: रन करने के तुरंत बाद वहां एक `.class` बाइटकोड फाइल भी अपने-आप बन जाएगी, जो हमने पिछले चैप्टर में पढ़ा था!)*
