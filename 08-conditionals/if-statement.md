# 🚦 The `if` Statement in Java

## 🤔 What is the `if` Statement?
प्रोग्रामिंग में हमारी मेन रिस्पांसिबिलिटी (Responsibility) कोड लिखना होती है जो सिचुएशन के हिसाब से सही डिसीजन (Decision) ले सके। 

**`if` statement** जावा का सबसे बेसिक और सिंपल कंडीशनल कंस्ट्रक्ट है। यह प्रोग्राम को बताता है कि:
> *"सिर्फ़ तभी इस कोड को रन करो, जब दी गई Condition (शर्त) **TRUE (सही)** हो।"*

अगर कंडीशन **FALSE (गलत)** होती है, तो जावा उस कोड को पूरी तरह से इग्नोर कर देता है और आगे बढ़ जाता है (हमें कोई भी आउटपुट नहीं मिलता)।

---

## 👨‍👦 "The Ram & ₹10" Analogy
मान लीजिए कि एक लड़का है राम, और उसके पास कुछ पैसे हैं।
- **Condition:** क्या राम के पास ₹10 हैं?
- **True:** अगर उसके पास ₹10 हैं, तो आउटपुट आएगा *"Yes, Ram is having ₹10"*.
- **False:** अगर उसके पास ₹11 हैं या ₹9 हैं, तो कंडीशन फेल हो जाएगी। प्रोग्राम बिना कुछ बोले चुपचाप आगे बढ़ जाएगा!

---

## 🛠️ Syntax of `if` Statement

```java
if (condition) {
    // यह कोड सिर्फ़ तभी रन होगा जब Condition True होगी
}
```
1. सबसे पहले `if` कीवर्ड लिखते हैं।
2. फिर Round Brackets `()` के अंदर **Condition** देते हैं (जैसे `age >= 18`)।
3. फिर Curly Brackets `{}` के अंदर वह कोड लिखते हैं जो हमें एग्जीक्यूट (Execute) करवाना है।

## 💻 Real World Example (Voting System)

मान लीजिए हम चेक कर रहे हैं कि कोई यूज़र वोट देने के लिए एलिजिबल है या नहीं।

### Scenario 1: Condition is True
```java
int age = 18;

if (age >= 18) {
    System.out.println("You can vote");
}
// Output: You can vote
```
यहाँ `age` की वैल्यू `18` है। कंडीशन `18 >= 18` **True** है, इसलिए ब्रैकेट के अंदर का मैसेज प्रिंट हो गया।

### Scenario 2: Condition is False
```java
int age = 17;

if (age >= 18) {
    System.out.println("You can vote"); 
}
// Output: (Empty - Nothing happens!)
```
यहाँ `age` की वैल्यू `17` है। कंडीशन `17 >= 18` **False** है। जैसे ही जावा ने देखा कि कंडीशन गलत है, वो Curly Brackets `{}` के अंदर गया ही नहीं। हमारा प्रोग्राम चुपचाप आगे बढ़ गया और स्क्रीन पर कुछ भी आउटपुट नहीं आया।
